HTML Definitions

This page contains a descriptive list of common HTML terms.

HTML:
HyperText Markup Language â€“ the standard language for creating web pages.
Tag:
A keyword enclosed in angle brackets used to define elements in HTML.
  • Paired tag
  • Un paired tag
Element:
A complete structure in HTML consisting of an opening tag, content, and a closing tag.
Attribute:
Additional information added to an HTML element, usually inside the opening tag.
Head:
The part of the HTML document containing meta information, title, and links to styles/scripts.
Body:
The main content area of the HTML document that is displayed in the browser.
Doctype:
A declaration at the top of an HTML document that tells the browser which HTML version to use.
Anchor:
An HTML element (<a>) used to create hyperlinks.
List:
A group of related items represented using ordered, unordered, or descriptive lists.
